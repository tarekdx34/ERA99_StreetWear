QUTB — SALT JOURNAL (Editorial Magazine Page Prompt)
You are a world-class fashion editor, UX designer, and frontend engineer.
Design and build “SALT JOURNAL”, an editorial magazine experience for the brand QUTB.
This is not a blog and not a news feed. It is a slow, curated fashion journal where each entry feels like a printed editorial spread translated into a digital luxury reading experience.
________________________________________
Brand Context (critical)
QUTB identity:
•	Premium Egyptian basics label 
•	Aesthetic: TOTEME restraint × Jacquemus softness × Aimé Leon Dore warmth 
•	Calm luxury, editorial fashion tone 
•	Clothing framed as culture, memory, and atmosphere 
Visual system:
•	Background: bone / off-white dominant 
•	Text: soft charcoal / washed black 
•	Accents: muted beige, faded sand, warm gray 
•	Typography: editorial serif for titles + clean grotesk for body 
•	Layout: magazine grid, generous spacing, asymmetry allowed 
Tone:
•	Poetic but controlled 
•	Observational, not promotional 
•	No hype language 
•	No product pushing 
________________________________________
Concept
Title: SALT JOURNAL
Subtitle:
“Notes on fabric, form, and everyday culture.”
The page should feel like:
•	A printed fashion journal (System, Gentlewoman, Apartamento influence) 
•	A gallery of essays, short visual stories, and seasonal reflections 
•	A slow archive of QUTB worldview 
________________________________________
Page Structure
1. Journal Index (Landing Section)
A minimal editorial index of entries.
Layout:
•	Left: “SALT JOURNAL” 
•	Right: scrollable list of entries 
Each entry format:
•	Title 
•	Short one-line description 
•	Date (subtle, not dominant) 
Example entries:
•	“On the Weight of Summer Cotton” 
•	“The Silence of Neutral Colors” 
•	“Uniforms of the City” 
•	“Wearing Time, Not Trends” 
Interaction:
•	Hover reveals image preview (soft fade) 
•	Click opens full editorial article 
________________________________________
2. Featured Editorial Spread
Top story displayed as a full editorial layout.
Structure:
•	Full-width image or diptych 
•	Large serif headline 
•	Short opening paragraph 
•	Pull quotes embedded in layout 
Layout style:
•	Asymmetrical magazine grid 
•	Text wrapping around imagery 
•	Controlled negative space 
Mood:
•	High fashion editorial, but quiet and wearable 
________________________________________
3. Article Page Template (Core Reading Experience)
Each journal entry opens as a long-form editorial scroll.
Sections:
Header
•	Title (large serif) 
•	Subtitle (small caps) 
•	Date (minimal) 
________________________________________
Body Content Blocks
Use mixed layout blocks:
•	Paragraph text (narrow column width) 
•	Full-bleed imagery 
•	Split image + text sections 
•	Pull quotes in oversized serif 
•	Section dividers (thin lines or spacing only) 
________________________________________
Writing Style Rules:
•	No marketing tone 
•	No technical fabric jargon 
•	Focus on atmosphere, memory, behavior, and culture 
•	Clothing described through life moments, not specs 
Example tone:
•	“Cotton is not a material. It is a daily rhythm.” 
•	“Neutral tones do not speak loudly. They stay longer.” 
________________________________________
4. Visual Essays Section
A separate section for image-led stories.
Format:
•	Grid of 2–3 columns (asymmetrical allowed) 
•	Each tile: 
o	Image 
o	Title overlay (minimal) 
o	One-line caption 
Content types:
•	Street observations 
•	Fabric close-ups 
•	Seasonal moods 
•	City silhouettes in QUTB garments 
Interaction:
•	Slow zoom on hover 
•	Fade transitions only 
________________________________________
5. Categories (Sub-Journal Filters)
Minimal filter system:
•	Fabric Notes 
•	Seasonal Studies 
•	City Observations 
•	Uniform Studies 
•	Material Essays 
Design:
•	Horizontal scroll or subtle top bar 
•	No heavy UI borders 
________________________________________
6. Closing Section — Archive Philosophy
Final statement block:
“SALT JOURNAL is not content.
It is observation.”
Secondary line:
“What we wear is what we return to.”
End state:
•	Bone background fade 
•	Minimal QUTB wordmark 
________________________________________
Interaction Rules
•	Slow editorial transitions only 
•	No aggressive motion or parallax systems 
•	Hover = soft fade + slight image reveal 
•	Scroll = narrative progression, not UI navigation 
•	Mobile = linear magazine reading flow 
•	Priority: readability + spacing + calm pacing 
________________________________________
Typography System
•	Titles: refined serif (editorial fashion magazine tone) 
•	Body: clean grotesk (light weight, wide tracking) 
•	Captions: small caps, low contrast 
Hierarchy must feel like:
•	Magazine spread, not website UI 
________________________________________
Technical Direction
•	CSS grid-based editorial layout 
•	Image-first lazy loading 
•	Soft transitions (opacity + transform only) 
•	Strict spacing system (8pt or 12pt grid) 
•	No borders unless extremely subtle 
•	Performance optimized for image-heavy reading 
________________________________________
Output Expectation
Generate:
•	Full page architecture 
•	Journal index system 
•	Article template system 
•	Visual essay grid system 
•	Interaction model 
•	Copy direction aligned with QUTB editorial identity
