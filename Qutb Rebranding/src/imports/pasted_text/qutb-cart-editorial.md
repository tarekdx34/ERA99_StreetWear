QUTB — CART EXPERIENCE (Editorial Checkout Prompt)
You are a world-class UX designer and fashion e-commerce creative director.
Design and build a “Cart” experience for the brand QUTB.
This is not a transactional checkout UI. It is a calm editorial holding space for selected garments.
________________________________________
Brand Context (critical)
QUTB:
•	Premium Egyptian basics brand 
•	Inspired by TOTEME / Jacquemus / Aimé Leon Dore 
•	Soft luxury, editorial restraint 
•	No urgency, no aggressive conversion language 
Cart philosophy:
•	Selection, not purchase pressure 
•	Clothing as considered composition 
•	A pause between desire and ownership 
________________________________________
Concept
Title:
“Your Selection”
Subtitle:
“A quiet inventory of what you’ve chosen.”
The cart should feel like:
•	A museum holding room for garments 
•	A fashion stylist’s selection board 
•	A calm pre-purchase editorial page 
________________________________________
Page Structure
1. Cart Header (Minimal)
•	“YOUR SELECTION” 
•	Small subline: “Pieces held for consideration” 
•	No totals emphasis upfront 
________________________________________
2. Item List (Core Section)
Each cart item is an editorial object, not a product row.
Each item layout:
•	Left: garment image (soft, full bleed crop) 
•	Right: 
o	Name (serif or refined grotesk) 
o	Color (muted descriptor, not SKU language) 
o	Size (minimal) 
o	Price (quiet, not dominant) 
Interactions:
•	Hover: slow image reveal or fabric detail zoom 
•	Remove action: subtle “fade out” not delete animation 
No borders, no heavy dividers.
Spacing defines structure.
________________________________________
3. Styling Notes (Optional Section)
For each item:
•	“Styling note” (optional line) 
•	Example: 
o	“Best worn with structured trousers in neutral tones.” 
o	“Works as a foundational layer for colder evenings.” 
Tone:
•	Editorial styling advice, not upselling 
________________________________________
4. Summary (Quiet Total Section)
Structure:
•	Subtotal 
•	Shipping note (minimal, region-based if needed) 
•	Total 
Design rules:
•	Small typography 
•	No aggressive hierarchy 
•	No bright emphasis colors 
________________________________________
5. Action Section (Checkout)
Primary action:
“Continue”
Not:
•	“Buy now” 
•	“Checkout” 
Secondary action:
“Return to collection”
Design:
•	Minimal button, outlined or soft filled bone/charcoal contrast 
•	No urgency cues 
•	No countdowns, no scarcity 
________________________________________
Interaction Rules
•	No animations faster than 400–600ms 
•	No bounce or elastic motion 
•	Soft fade transitions only 
•	Mobile: vertical stacked list with sticky bottom summary 
•	No persistent cart noise or badges 
________________________________________
Visual Language
•	Background: bone/off-white 
•	Text: soft black/charcoal 
•	Accents: muted beige or ash gray 
•	Typography: editorial serif + clean grotesk pairing 
•	Layout: magazine-like spacing, not grid-heavy commerce UI 
________________________________________
Output Expectation
Generate:
•	Cart page layout 
•	Item component system 
•	Summary module 
•	Checkout interaction flow 
•	Mobile adaptation rules 
________________________________________

QUTB — ALEXANDRIA PAGE (CITY EDITORIAL PROMPT)
You are a world-class fashion editor and experiential web designer.
Design and build an editorial city page titled:
“ALEXANDRIA”
This is not a travel page. It is a city as atmosphere, fabric, and cultural memory interpreted through QUTB clothing.
________________________________________
Brand Context (critical)
QUTB positioning:
•	Egyptian-rooted luxury basics label 
•	Calm, international editorial tone 
•	Inspired by Mediterranean softness + European minimal fashion houses 
Alexandria relevance:
•	Origin city 
•	Coastal light, humidity, salt air, stone textures, faded colors 
•	Emotional reference point for brand identity 
________________________________________
Concept
Title:
“ALEXANDRIA”
Subtitle:
“Light, salt, and slow permanence.”
This page should feel like:
•	A coastal editorial shoot 
•	A memory archive of a Mediterranean city 
•	A fashion campaign grounded in place 
________________________________________
Page Structure
1. Hero — Coastal Stillness
Full screen:
•	Soft, washed coastal imagery (sea, stone, early morning light) 
•	Large serif title: “ALEXANDRIA” 
•	Subline:
“A city written in salt and light.” 
Motion:
•	Extremely slow fade-in 
•	Minimal movement, almost static 
________________________________________
2. City Atmosphere Section
Describe city through sensory language:
Focus areas:
•	Sea humidity and salt air 
•	Limestone and faded architecture 
•	Sun-washed tones 
•	Slow coastal rhythm 
Layout:
•	Full-width image blocks alternating with text 
•	No rigid grid symmetry 
Tone:
•	Observational, cinematic, restrained 
________________________________________
3. Wardrobe in Context
Show QUTB clothing in Alexandria environment.
Content:
•	Oversized cotton in coastal wind 
•	Neutral tones against stone and sea 
•	Clothing interacting with environment, not posing 
UI:
•	Editorial lookbook strips 
•	Soft hover zoom 
•	No product labeling dominance 
________________________________________
4. Color Study of the City
Translate Alexandria into palette:
•	Salt white 
•	Faded blue 
•	Sun-washed beige 
•	Concrete gray 
•	Deep coastal navy 
Presentation:
•	Horizontal color fields or stacked swatches 
•	No technical naming, purely atmospheric 
________________________________________
5. Movement Section — Slow Life
Describe lifestyle rhythm:
•	Walking 
•	Waiting 
•	Sitting near sea 
•	Quiet daily repetition 
Visual:
•	Minimal video loops or still image sequences 
•	No fast cuts 
________________________________________
6. Closing Statement
Final text:
“A city is not visited. It is absorbed.”
Secondary line:
“Alexandria remains in fabric long after it leaves the body.”
End:
•	Fade to bone background 
•	QUTB wordmark 
________________________________________
Interaction Rules
•	Scroll = cinematic progression 
•	No UI urgency 
•	No interactive clutter 
•	Motion must feel like natural time, not interface animation 
•	Mobile = linear editorial story 
________________________________________
Output Expectation
Generate:
•	Full page layout 
•	Section structure 
•	Copywriting blocks 
•	Visual direction system 
•	Interaction model 
•	Mobile adaptation rules
